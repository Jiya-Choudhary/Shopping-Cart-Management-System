# 🛒 Shopping Cart Management System

A Python-based shopping cart application that allows users to add, remove, and manage items while calculating the total bill in real time.

## ✨ Features

* ➕ Add New Items
* 🗑️ Remove Items
* 💰 Calculate Total Bill
* 📦 Manage Product List
* ⚡ Interactive Menu-Driven Interface

## 🛠️ Tech Stack

* Python 3

## 🚀 How to Run

```bash
python shopping_cart.py
```

## 📌 Example Output

```text
Your Shopping Cart

You Added :
{'Trouser': 2222, 'SoftToy': 3456, 'Bracelet': 1286}

Enter option: Total Bill

Total Bill is: 6964
```
